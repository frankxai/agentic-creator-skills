## Prompt Intensifiers

Add these to any prompt to boost quality:

| Intensifier | Effect |
|-------------|--------|
| `cinematic color grading` | Film-like, moody |
| `subsurface scattering` | Realistic materials |
| `editorial composition` | Professional, intentional |
| `8K quality, highly detailed` | Sharpness and detail |
| `depth of field` | Foreground/background separation |
| `ray tracing, realistic lighting` | Photorealistic |
| `Unreal Engine 5 render` | High-end 3D look |
| `atmospheric perspective` | Depth and scale |

## Universal Negatives

Add to every prompt:

```
no text, no watermarks, no logos, no blur on subject, no overexposure, no flat lighting, no cartoon style, no clipart, no chrome robot, no single-color saturation, no lens flare
```
