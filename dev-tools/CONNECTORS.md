# Dev Tools Connectors

| Category | Default | Notes |
|----------|---------|-------|
| `~~shell` | bash/zsh | All tools require shell access |
| `~~filesystem` | Local | Scans local filesystem only |

## No External Connectors Required

All dev-tools skills operate entirely on the local filesystem. No APIs, no credentials, no external services.

## Standalone CLI

The storage-intelligence skill is also available as a standalone npm package:
```bash
npx storage-intelligence
```

Source: [github.com/frankxai/storage-intelligence](https://github.com/frankxai/storage-intelligence)
