# Product Launcher Connectors

| Category | Default | Alternatives |
|----------|---------|-------------|
| `~~publishing` | Vercel (git push → auto-deploy) | Railway, Render, Netlify, Fly.io |
| `~~knowledge base` | Local docs/ + spec files | Notion, Confluence, Linear |
| `~~task management` | Local TASKS.md | Linear, GitHub Issues, Notion |
| `~~analytics` | Vercel Analytics | PostHog, Plausible |
| `~~payment` | Lemon Squeezy | Stripe, Paddle, Gumroad |
| `~~email marketing` | Resend | ConvertKit, Beehiiv |
| `~~chat` | Slack | Discord, Teams |

## Connector Usage in This Plugin

- `~~publishing` — deploy product to production via `/product-deploy`
- `~~knowledge base` — store and retrieve product specs, PRDs, architecture decisions
- `~~task management` — sync sprint tasks with external project tool
- `~~analytics` — pull performance data for post-launch retro
- `~~payment` — create checkout links, manage pricing for digital products
- `~~email marketing` — notify waitlist or subscriber list on launch
- `~~chat` — send launch announcements, deploy notifications, sprint summaries
