# File Templates

Standard templates for initializing the creator productivity workspace.

## TASKS.md Template

```markdown
# Tasks

_Last updated: [date]_

## In Progress
<!-- Currently being worked on -->

## Active
<!-- Queued, ready to start -->

## Waiting On
<!-- Blocked or waiting for external input -->

## Done (this week)
<!-- Move items here on completion, archive weekly -->
```

- [ ] [Deliverable 2]
